== Description ==
VG Block REST API is block the access to the REST API for the normal user and register user, only Admin can access the rest API.

== Feature ==

* Block the REST API.
* Secure the data from users.
* Block the JSon data.

== Installation ==
*	Download the plugin .zip file.
*	Login in to admin Click Plugins -> Add New -> Upload.
*	Find VG Remove html comment in plugin list  and activate it.
